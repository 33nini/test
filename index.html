На данном сайте студентам предоставлен курс обучения по диспциплине «Математические методы». Все темы курса предоставлены в виде четырех разделов. Для получения оценки по окончанию курса студенты должны изучить теоретический материал, выполнить лабораторные работы, а также самостоятельные работы, которые будут прикреплены к теоретическому материалу.

1 раздел

  • Тема 1.1. Введение. Моделирование как метод познания.
  • Тема 1.2. Этапы построения математической модели задачи. Математическое программирование.

2 раздел

  • Тема 2.1. Постановка задачи линейного программирования. Графический метод решения ЗЛП.
  • Тема 2.2. Симплекс метод решения задач линейного программирования.
  • Тема 2.3. Целочисленное программирование.
  • Тема 2.4. Транспортная задача линейного программирования.

3 раздел

  • Тема 3.1. Технология имитационного моделирования.

4 раздел

  • Тема 4.1. Понятие прогноза. Количественные методы прогнозирования.
